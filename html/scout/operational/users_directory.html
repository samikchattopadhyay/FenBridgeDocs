Registered Users Directory

Browse and manage all farmers and poppers scout registered or is assigned to. Supports search/filter by role and status. Detailed profiles with inline edit actions for identity, ponds, bank accounts, and documents.

Directory Features & User Hub Pattern

Central Hub
Users Directory is the scout's on-behalf management hub. One-stop for registering, editing, and monitoring farmers/poppers.
Inline Actions
Quick actions (edit identity, add pond, update bank) directly from detail tabs. No separate edit screens for most actions.
Status Visibility
Pending-approval users clearly marked. Activity feed shows recent transactions (bids, verifications, payments) for accountability.
Offline-First
User list and profiles cached locally. Edits captured offline; sync via sync_queue when network available.
