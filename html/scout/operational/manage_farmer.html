Workflow Module 5

Manage Farmer

Edit and manage registered farmer profiles. Supports identity updates, pond management (add/edit/deactivate), bank account changes with penny-drop verification, document uploads, and preference management.

Farmer Management Actions
8 Actions
1. Edit Identity
Update farmer's name, address, contact info.
Editable Fields
  • Full name
  • Address (district, village, free-text)
  • Alternate phone
Requirements
  • Name changes allowed by scout
  • Mobile changes require OTP re-verification
2. Add Pond
Register new pond for farmer (4-step mini-wizard).
Steps
  • Pond details (name, area, capacity)
  • Address (district, village)
  • Geo-fencing (GPS capture)
  • Legal documents
Status
  • Added pond enters pending_approval until Admin approves
3. Edit Pond
Update existing pond details.
Editable Fields
  • Pond name, area, capacity
  • Address details
  • GPS location
Restrictions
  • Pond code immutable
  • GPS edits flagged for Admin review
4. Deactivate Pond
Mark pond inactive (farmer no longer farms it).
Process
  • Select coded reason (Closed, Leased Out, Defunct, Other)
  • Soft deactivation (not deleted)
Effect
  • Pond unavailable for new lot registrations
5. Add / Replace Bank Account
Add or replace farmer's bank account for payouts.
Process
  • Account holder name, account number (confirmed), IFSC
  • Penny-drop verification triggered at submit
Policy
  • One active account per farmer (Year 1)
  • Old account marked inactive
  • Penny-drop re-runs automatically
6. Initiate / Retry Penny-Drop
Re-trigger penny-drop verification if previous attempt failed.
Trigger
  • Scout can manually retry failed penny-drops
Escalation
  • Persistent failures escalate to Admin for manual review
7. Add / Replace Identity Documents
Upload or replace farmer's identity documents.
Document Types
  • Aadhar / PAN / Voter ID / Other
  • Up to 3 photos per document
Purpose
  • Supports compliance and farmer profile verification
8. Update Language / Theme on Behalf
Change farmer's app preferences.
Options
  • Language: English / Hindi / Bengali
  • Theme: Dark / Light

Management Pattern & Approval Gates

Sensitive Changes
Mobile number changes require OTP re-verification. Name changes scout-allowed. GPS edits flagged for Admin review.
Pond Management
New ponds enter pending_approval. Farmers can have multiple ponds; deactivation is soft (not permanent deletion).
Bank Account Policy
One active account per farmer (Year 1). Old accounts marked inactive. Penny-drop is mandatory for payout eligibility.
Offline-First
All edits captured locally; documents compressed. Sync via sync_queue when network available.
