Workflow Module 6

Mark Lot Ready

5-step wizard for signalling production completion to buyer. Farmer/popper confirms bagged quantity and bag count with mandatory photos. Notifies Admin and schedules scout verification task before lot can be dispatched.

Ready-to-Dispatch Steps
5 Steps
Step 1Choose Active PO
Purpose
Select the in-progress purchase order to mark as ready.
Filter & Display
Only in_progress POs awarded to user shown. Cards show buyer name, product spec, quantity, bidding details.
Step 2Confirm Quantity Bagged
Purpose
Capture actually bagged quantity (kg) produced by farmer/popper.
Input
Numeric field (kg). Soft warning if differs >5% from PO quantity.
Requirement
Captures actual production yield; may differ from PO contracted qty due to weather, waste, etc.
Step 3Confirm Bag Count
Purpose
Capture number of bags in which lot is packed.
Input
Numeric field (bag count). E.g., "25 bags" for 500kg lot.
Step 4Evidence Photos (Mandatory)
Purpose
Visual proof of bagged lot ready for dispatch.
Required Photos
  • Wide shot (showing all bags, lot scale)
  • Close-up (showing bag detail, labelling if present)
Requirement
Minimum 2 photos mandatory. Cannot proceed without both. Compress â‰¤1.5MB each; GPS-stamped.
Step 5Notify Admin & Confirmation
Purpose
Submit ready signal, notify Admin, schedule scout verification task.
What's Shown
  • Final recap (PO details, bagged qty, bag count, photos)
  • Confirm checkbox before submit
