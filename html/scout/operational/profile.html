Workflow Module 7

Profile & Settings

Scout account management and preferences. View/edit profile, manage assigned areas, language/theme selection, offline mode status, sync information, and security/logout options.

Profile & Settings Options
6 Sections
1. Profile Information
View and edit scout's personal account details.
Viewable Fields
  • Scout full name
  • Scout ID (generated, immutable)
  • Email (if linked)
  • Mobile number (primary contact)
Editable Fields
  • Scout name (allow minor edits)
  • Alternate phone (optional)
2. Update Mobile Number
Change primary mobile contact with OTP verification.
Process
  • Scout enters new mobile
  • OTP sent to new number
  • Scout enters OTP to confirm ownership
Outcome
  • Mobile updated in system; all future comms sent to new number
3. Assigned Geographic Areas
View geographic territories scout is assigned to.
Display
  • Read-only list of districts and sub-districts assigned
  • Assignment controls visibility in all directory screens
  • Cannot be edited by scout (admin-controlled)
Scope Impact
  • Ponds Directory: only scout-assigned ponds visible
  • Warehouses Directory: only scout-assigned warehouses visible
  • Users Directory: only scout-registered or scout-assigned users visible
4. Language & Theme Preferences
Customize scout's app language and display theme.
Language Options
  • English (default)
  • Hindi
  • Bengali
Theme Options
  • Dark (default, optimized for field usage)
  • Light
Scope
  • Preference persisted locally; applies to all screens
5. Offline Mode & Sync Information
Monitor offline status, sync state, and manual sync trigger.
Offline Mode
  • Status indicator: Online / Offline (system-managed, read-only)
  • Shows when device detected network loss
Sync Information
  • Last successful sync timestamp
  • Pending items count (awaiting upload)
  • "Sync Now" button to trigger manual sync
  • Sync progress indicator during active sync
Network Requirements
  • Sync requires WiFi or mobile data connection
  • User can work fully offline; sync happens on demand
6. Security & Account Management
Manage account security, password, and logout.
Account Info
  • Scout ID display
  • Account creation date
  • Last login timestamp
Password Management
  • Optional password change (recommended every 90 days)
  • Requires current password confirmation
  • New password strength validation (minimum 8 chars, mixed case, numbers)
Logout
  • Logout button clears local session
  • Pending sync items saved locally; resume sync after re-login

Profile Management & Account Security

Data Privacy
Scout's personal data (name, mobile) is private. Profile edits captured locally. No sharing of scout identity with users unless explicitly required by workflow.
Geographic Scoping
Assigned areas control all directory visibility and task routing. Admin-managed; scout read-only. Reassignments sync automatically to device.
Offline-First Sync
All settings cached locally for offline access. Profile/language/theme persisted across sessions. Sync information updates when network available.
Security & Logout
Logout clears session but preserves sync queue locally. Scout can resume work after re-login. Password changes recommended every 90 days for security.
