Workflow Module 5

Manage Popper

Edit and manage registered popper business profiles. Supports business details updates, warehouse management (add/edit/deactivate), bank account changes with penny-drop verification, document uploads, and preference management.

Popper Management Actions
9 Actions
1. Edit Business Details
Update popper business name, GSTIN, FSSAI, contact info.
Editable Fields
  • Business name
  • GSTIN
  • FSSAI license number
  • Contact person & alternate phone
Requirements
  • Business name changes allowed by scout
  • GSTIN/FSSAI changes flagged for Admin review
2. Add Warehouse
Register new warehouse for popper (4-step mini-wizard).
Steps
  • Warehouse details (name, capacity, type)
  • Address (district, village, GPS)
  • Fitness assessment (initial baseline)
  • Legal documents
Status
  • Added warehouse enters pending_approval until Admin approves
3. Edit Warehouse
Update existing warehouse details.
Editable Fields
  • Warehouse name, capacity, type
  • Address details
  • GPS location
Restrictions
  • Warehouse code immutable
  • GPS edits flagged for Admin review
4. Deactivate Warehouse
Mark warehouse inactive (popper no longer operates).
Process
  • Select coded reason (Closed, Transferred, Defunct, Other)
  • Soft deactivation (not deleted)
Effect
  • Warehouse unavailable for lot sealing and storage
5. Add / Replace Bank Account
Add or replace popper's bank account for payouts.
Process
  • Account holder name, account number (confirmed), IFSC
  • Penny-drop verification triggered at submit
Policy
  • One active account per popper (Year 1)
  • Old account marked inactive
  • Penny-drop re-runs automatically
6. Initiate / Retry Penny-Drop
Re-trigger penny-drop verification if previous attempt failed.
Trigger
  • Scout can manually retry failed penny-drops
Escalation
  • Persistent failures escalate to Admin for manual review
7. Upload Business Documents
Upload or replace popper business documents.
Document Types
  • GSTIN / FSSAI / Business Registration / Factory License
  • Up to 3 photos per document type
Purpose
  • Supports compliance and business legitimacy verification
8. Upload Warehouse Documents
Upload warehouse-specific legal and operational documents.
Document Types
  • Warehouse lease / ownership deed / fitness checklist
  • Up to 3 photos per warehouse document type
9. Update Language / Theme on Behalf
Change popper's app preferences.
Options
  • Language: English / Hindi / Bengali
  • Theme: Dark / Light

Popper Management Pattern & Approval Gates

Sensitive Changes
GSTIN/FSSAI changes flagged for Admin review. Business name scout-allowed. GPS edits flagged for Admin.
Warehouse Management
New warehouses enter pending_approval. Poppers can have multiple warehouses; deactivation is soft (not permanent deletion).
Bank Account Policy
One active account per popper (Year 1). Old accounts marked inactive. Penny-drop is mandatory for payout eligibility.
Offline-First
All edits captured locally; documents compressed. Sync via sync_queue when network available.
